# Aetherion Robotics

  We are two students passionate about robotics, embedded systems, electronics, and intelligent hardware design. Our projects focus on building real-world engineering systems, including robotics platforms, biomedical monitoring devices, autonomous systems, and electronic prototypes.
  We enjoy combining software, electronics, and mechanical design to create innovative and practical solutions while continuously improving our engineering and problem-solving skills.

---

## Projects

###  Bluetooth Car
Arduino-based robotic car controlled wirelessly via Bluetooth from a mobile device. The system enables manual navigation and motor control, allowing forward, backward, and turning movements. This project focuses on understanding embedded systems, motor drivers, and wireless communication.
 [Robot 1 - Bluetooth Car](./bluetooth_car)

https://github.com/KissCezar/Bluetooth-Car

---

###  Drone System
A manually controlled drone operated using a handheld RC transmitter. The project focuses on flight stability, control precision, and piloting skills using a basic drone platform with brushless motors and ESC controllers.

https://github.com/KissCezar/Drone

---

###  EKG Monitor
A biomedical monitoring system designed to capture and display heart electrical signals in real time. The project uses sensors and signal processing techniques to visualize heartbeat activity and understand basic medical data acquisition systems.

https://github.com/KissCezar/EKG

---

###  Robotic Hand
A servo-motor based robotic hand designed to replicate basic human finger movements. The system is controlled using a microcontroller and demonstrates principles of mechanical design, actuator control, and embedded programming.

https://github.com/KissCezar/Robotic-Hand

---

###  Drowsiness Detection System
A sensor-based system designed to detect driver fatigue by monitoring eye behavior and facial signals. When signs of drowsiness are detected, the system triggers an alert to improve driver safety. This project explores basic computer vision and real-time sensor processing concepts.

https://github.com/yourname/Drowsiness-Detection-System

---
###  Tesla Coil

A high-voltage Tesla coil project designed to generate electrical discharges and demonstrate wireless energy transfer through electromagnetic field principles. The system explores resonance, coil coupling, and high-frequency voltage transformation for experimental electronics learning.

https://github.com/KissCezar/Tesla-Coil

---

We aim to develop practical engineering systems that integrate robotics, embedded systems, and electronics to solve real-world problems. Through our projects, we focus on learning hardware design, programming, and system integration.
